# vfox-mailpit

[![CI](https://github.com/jbox-web/vfox-mailpit/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-mailpit/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [mailpit](https://github.com/axllent/mailpit)

## Install

```
mise plugin install mailpit https://github.com/jbox-web/vfox-mailpit.git
```
